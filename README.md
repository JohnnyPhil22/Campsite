## How to use it:
### Linux:
1) Clone or download this repository
2) Go to Terminal and go to the folder where saved
3) To start:
#### Ubuntu:
python3 app.py
#### Manjaro:
app python3 app.py
### Windows
1) Clone or download this repository
2) Go to Command Prompt and go to folder where saved
4) Type py appforwindows.py
